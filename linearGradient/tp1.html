g the so-called color stops. Breakpoints are set in %, where 0% is the starting point 100% is the ending point, for example: .grad4 { height: 200px; background: linear-gradient(to top, red 20%, #E4E4D8 50%, #A19887 80%);/* def color red stopper a 20% ....*/ }

For a clear distribution of colored bands each subsequent color should start from the point it stopped the previous color:

g background: linear-gradient(to right, #FFDDD6 20%, #FFF9ED 20%, #FFF9ED 80%, #DBDBDB 80%);

Radial gradient radial-gradient()

g The shape of the gradient is determined by key words circle or ellipse. If the form is not set, the default radial gradient takes the form of an ellipse. .grad6 { height: 200px; background: radial-gradient(white, #FFA9A1); }

The position of the center is specified using the keyword used in the property background-position, with the addition of consoles at. If the position of the center is not set, use default value at center.

g .grad7 { height: 200px; background: radial-gradient(at top, #FEFFFF, #A7CECC); }
